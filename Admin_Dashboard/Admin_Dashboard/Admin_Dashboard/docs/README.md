## Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No server setup required - runs entirely in the browser

### Installation
1. Clone or download the Admin_Dashboard folder
2. Open `index.html` in your web browser
3. Use the default credentials to login:
   - **Username:** `admin`
   - **Password:** `admin123`

### File Structure
```
Admin_Dashboard/
├── index.html          # Main dashboard page
├── users.html          # User management page
├── styles.css          # Main stylesheet
├── pages.css           # Page-specific styles
├── script.js           # Main JavaScript functionality
├── pages.js            # Page-specific JavaScript
└── README.md           # This file
```

## Usage

### Login
1. Open `index.html` in your browser
2. Enter credentials (admin/admin123)
3. Click "Sign In" to access the dashboard

### Navigation
- Use the sidebar to navigate between different sections
- Click on menu items to switch pages
- Use the search bar to find specific content
- Access user menu for account options

### User Management
1. Navigate to "Users" in the sidebar
2. Use filters to find specific users
3. Click "Add User" to create new users
4. Use action buttons to view, edit, or delete users

### Responsive Design
- Fully responsive design works on all devices
- Mobile-optimized interface
- Touch-friendly controls
- Adaptive layout

## Customization

### Styling
- Modify `styles.css` for global styles
- Update `pages.css` for page-specific styles
- Customize colors, fonts, and layout as needed

### Functionality
- Extend `script.js` for main functionality
- Modify `pages.js` for page-specific features
- Add new pages by following the existing structure

### Adding New Pages
1. Create new HTML file following the structure of `users.html`
2. Add corresponding CSS styles
3. Implement JavaScript functionality
4. Update navigation menu
